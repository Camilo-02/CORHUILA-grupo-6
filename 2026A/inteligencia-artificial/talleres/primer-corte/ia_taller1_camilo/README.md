# Taller 1 - Inteligencia Artificial

Implementación desde cero de:

1. Perceptrón simple
2. Red neuronal multicapa (MLP) con Backpropagation

## Requisitos

- Python 3.10 o superior
- NumPy
- Matplotlib
- Pandas

## Estructura

- `perceptron_simple.py`: implementa un perceptrón simple para dos datasets linealmente separables.
- `mlp_xor.py`: implementa una MLP desde cero para resolver XOR.
- `resultados/`: almacena las gráficas generadas al ejecutar los scripts.

## Instalación

```bash
pip install -r requirements.txt
```

## Ejecución

```bash
python perceptron_simple.py
python mlp_xor.py
```
